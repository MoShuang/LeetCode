There are implement codes for the questions in Leetcode.

1. 格式规范
 - 文件名为Q问题序号_问题名.不同语言后缀
 - 例如Q338_CountingBits.py
 - 文件注释应包括算法思想、时间、空间复杂度以及具体运行时间（sample或submit）
 - 代码内注释：
     1. 单行
`1.
 #python 
2.
 //CPP
`
     2. 多行
```1.
   " " "
   Python
   " " "
2.
   /*
   CPP
   */
```

 - 另外注意在markdown的代码块用的```是tab上的键
